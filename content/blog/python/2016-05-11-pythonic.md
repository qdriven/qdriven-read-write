# Pythonic Style

Pythonic 是种很有趣的风格，有些实际上python的惯用方法和特有方法。

## Switch Two variables

python 通过tuple进行pack，unpack就不需要temp变量
```python
a,b=b,a
```
如果其他语言需要swap的话，就需要一个temp变量，然后再置换。

## P VS NP

```python
# P
a=3
b=1

1<=b<=a and a<10 # return True

## NP
b>=1 and b<=a and a<10 # True

```

```python
numList = [1,2,3,4,5]

from operator import mul
# reduce is interesting

# NP
sum = 0
maxNum = -float('inf')
minNum = float('inf')
prod = 1
for num in numList:
    if num > maxNum:
        maxNum = num
    if num < minNum:
        minNum = num
    sum += num
    prod *= num
# sum = 15 maxNum = 5 minNum = 1 prod = 120
```

## 列表推导式

```python
l = [x*x for x in range(10) if x % 3 == 0]
#l = [0, 9, 36, 81]

##NP
l = []
for x in range(10):
    if x % 3 == 0:
        l.append(x*x)
#l = [0, 9, 36, 81]

```
