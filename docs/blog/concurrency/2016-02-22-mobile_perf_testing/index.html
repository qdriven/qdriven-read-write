  • Monkey
  • Memory Usage: 
      /system/build.prop
  dalvik.vm.heapstartsize=8m
  # 堆分配的初始大小
  dalvik.vm.heapgrowthlimit=64m

  • DDMS
  • adb shell top
  • 启动性能(首次，多次)
  • 流量
  • 应用占用量，cachesize，datasize，codesize
  • 电量
  • CPU: active, idel
  • memory leak, hprof, ddms dump hprof
  • GPU 过度绘制 1.蓝色1x过度绘制 2.绿色2x过度绘制 3.淡红色3x过度绘制 4.红色超过4x过度绘制

  • systrace
  • traceview
  • tracer for OpenGL ES
